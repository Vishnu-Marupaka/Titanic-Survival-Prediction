# Titanic Survival Prediction 🚢

This project uses a Decision Tree Classifier to predict passenger survival based on data from the Titanic disaster. It serves as a foundational machine learning pipeline using Python and `scikit-learn`.

## Overview
The script loads passenger data, cleans and preprocesses the necessary features, and trains a machine learning model to predict whether a passenger survived or not. It then evaluates the model's accuracy on a holdout test set.

## Prerequisites
To run this code, you will need Python installed on your machine along with the following libraries:
* `pandas` (for data manipulation)
* `scikit-learn` (for building and evaluating the machine learning model)

You can install the required libraries using pip:
```bash
pip install pandas scikit-learn
